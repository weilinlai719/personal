# fireworks
# 🎆 Weilin's Interactive Fireworks Website

這是一個個人專案展示網頁，結合了 **HTML5 Canvas** 動態效果與現代化的 **Web UI 設計**。網頁背景會隨機綻放煙火，並支援滑鼠點擊互動，讓使用者在瀏覽時擁有沉浸式的慶典體驗。

## 🌐 專案預覽
你可以透過 GitHub Pages 直接觀看本專案：
👉 [點此開啟網頁](https://weilinlai719.github.io/fireworks/) 

## ✨ 核心功能
- **互動式煙火系統**：使用 JavaScript 粒子物理模擬，點擊畫面任何空白處即可施放煙火。
- **即時時鐘顯示**：網頁頂部設有玻璃質感的即時時鐘，精確顯示目前時間。
- **Taipei 101 導航**：畫面中央設有專屬「格子」連結，可快速跳轉至 Taipei 101 專屬煙火頁面。
- **響應式設計**：自動適應視窗大小，無論在電腦或手機瀏覽都能保持流暢動畫。

## 🛠️ 使用技術
- **前端語言**: HTML5, CSS3, JavaScript (ES6+)
- **繪圖技術**: Canvas API (粒子系統、重力模擬、殘影處理)
- **UI 風格**: Glassmorphism (毛玻璃質感)、CSS Flexbox/Absolute 定位

## 📁 檔案結構
- `index.html`: 專案核心檔案，包含所有的結構、樣式與動畫邏輯。
- `README.md`: 專案說明文件。

## 🚀 如何在本地運行
1. 下載或複製本專案代碼。
2. 使用瀏覽器直接開啟 `index.html`。
3. 建議使用 VS Code 的 **Live Server** 插件開發，以獲得最佳的即時預覽體驗。
